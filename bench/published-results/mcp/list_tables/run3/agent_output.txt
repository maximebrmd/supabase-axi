The public schema has **5 tables**:

| Table | Rows | RLS Enabled |
|---|---|---|
| `public.authors` | 4 | Yes |
| `public.posts` | 5 | Yes |
| `public.comments` | 5 | Yes |
| `public.tags` | 4 | No |
| `public.post_tags` | 7 | No |

**Security notice:** `public.tags` and `public.post_tags` have Row Level Security (RLS) disabled. This means they are fully exposed to anyone with the anon key. If you want to enable RLS on those tables, here is the SQL — but note that enabling RLS without also adding policies will block all access, so you'll need to define policies alongside it:

```sql
ALTER TABLE public.tags ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.post_tags ENABLE ROW LEVEL SECURITY;
```
