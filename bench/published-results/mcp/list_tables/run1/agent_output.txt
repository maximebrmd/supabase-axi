The public schema has **5 tables**:

| Table | Rows | RLS Enabled |
|---|---|---|
| `public.authors` | 4 | Yes |
| `public.posts` | 5 | Yes |
| `public.comments` | 5 | Yes |
| `public.tags` | 4 | No |
| `public.post_tags` | 7 | No |

**Security advisory:** `public.tags` and `public.post_tags` have Row Level Security (RLS) **disabled**, meaning any client with the anon key can read or modify every row. To enable it:

```sql
ALTER TABLE public.tags ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.post_tags ENABLE ROW LEVEL SECURITY;
```

Note that enabling RLS without adding policies will block all access to those tables, so you'll need to define appropriate policies before or alongside running this.
