The public schema has **5 tables**:

|---|---|---|
| `public.authors` | 4 | Yes |
| `public.posts` | 5 | Yes |

| `public.tags` | 4 | **No** |
| `public.post_tags` | 7 | **No** |

**Security advisory:** `public.tags` and `public.post_tags` have Row Level Security (RLS) disabled, meaning they are fully exposed to anyone with the anon key. If you want to enable RLS on them, here is the remediation SQL — but note you'll need to add policies afterward or all access will be blocked:

```sql
ALTER TABLE public.tags ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.post_tags ENABLE ROW LEVEL SECURITY;
```
